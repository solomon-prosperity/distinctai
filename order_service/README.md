## Description

Distinct AI Order Service is built with [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development - quick start with docker-compose
$ docker-compose -p distinctai up

# development
$ yarn run start

# watch mode
$ yarn run start:dev

# Orderion mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
